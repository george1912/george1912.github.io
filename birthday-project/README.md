# The Birthday Project

A one-page static birthday project that can be hosted on GitHub Pages.

## Replace these files

Put the final assets in the `assets` folder with these names:

- `pdf-one.pdf` for the front side
- `pdf-two.pdf` for the back side
- `reading-music.m4a`
- `author-image.png`
- `author-music.mp3`

Then open `index.html` in a browser, or publish the folder through GitHub Pages.
